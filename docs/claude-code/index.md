# Claude Code 客製化機制總覽

Claude Code 提供三種主要的客製化機制，讓你可以根據需求調整 Claude 的行為。

## 架構圖

```
┌─────────────────────────────────────────────────────────────┐
│              Claude Code 客製化機制架構                      │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────┐         │
│  │  CLAUDE.md  │  │    Slash    │  │    Skill    │         │
│  │             │  │   Command   │  │             │         │
│  │  📋 記憶    │  │  ⚡ 快速    │  │  🔧 能力    │         │
│  │  系統       │  │  指令       │  │  系統       │         │
│  └──────┬──────┘  └──────┬──────┘  └──────┬──────┘         │
│         │                │                │                 │
│         ▼                ▼                ▼                 │
│    ┌─────────┐      ┌─────────┐      ┌─────────┐           │
│    │自動載入 │      │手動觸發 │      │自動觸發 │           │
│    │每次對話 │      │ /cmd    │      │依請求   │           │
│    └─────────┘      └─────────┘      └─────────┘           │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 一句話定義

| 機制 | 定義 |
|------|------|
| **CLAUDE.md** | 🧠 「Claude 的記憶」- 每次對話都會載入的全局指令 |
| **Slash Command** | ⚡ 「快速提示」- 手動輸入 `/cmd` 觸發的預定義指令 |
| **Skill** | 🔧 「專業能力」- Claude 自動判斷並應用的複雜工作流 |

## 選擇決策樹

```
你想做什麼？
│
├─ 設定全局規則（每次都要生效）
│  └─ 使用 CLAUDE.md
│
├─ 建立快速指令（偶爾手動使用）
│  └─ 使用 Slash Command
│
└─ 自動化複雜流程（Claude 自動判斷）
   └─ 使用 Skill
```
